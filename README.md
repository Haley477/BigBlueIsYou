# BigBlueIsYou

**A puzzle game inspired by *Baba Is You*.**

BigBlueIsYou lets players manipulate the rules of the game world to solve puzzles. Change the way objects interact, experiment with the rules, and think creatively to complete each level.  

---

## About

BigBlueIsYou was developed as a final project for CS5410 by Haley Smith and Annie Houston. It is a collaborative puzzle game inspired by *Baba Is You*, where players can manipulate game rules to solve creative challenges.  

The game includes multiple levels with interactive objects, animations, particle effects, and sound feedback. Haley focused on implementing key gameplay mechanics, animations, particles, sound management, controls, and stability improvements to make the game polished and playable.

---

## Features

- Rule-based puzzle mechanics inspired by *Baba Is You*  
- Undo/Reset system for experimenting with rules  
- Animations for interactive game objects  
- Particle effects and sound feedback for game events  
- Flipping mechanics for entities  
- Intuitive controls and level transitions  
- Refactored code and project structure for maintainability  
- Designed to encourage creative problem-solving and experimentation

## Gameplay
Move your character around the level using arrow keys or WASD

Push words (like “Wall Is Stop” or “Blue Is You”) to change rules

Complete the level when your character reaches the goal according to the rules you set

Experiment! Many puzzles have multiple solutions

Use the Undo/Reset system to try different strategies
